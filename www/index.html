WikiRoom

WikiRoom vous permet de découvrir aléatoirement des articles de Wikipédia.

Cette application est complètement inspirée de WikiTok mais WikiRoom est garantie :

  • sans React,
  • sans Tailwind CSS,
  • sans utilisation d’IA,
  • sans système de build pour la fabriquer,
  • et sans OGM !

WikiRoom a été développé avec la librairie ultra légère Room et peut être installée (PWA) sur mobile ou sur ordinateur.

Le code source de WikiRoom est disponible sur GitHub.

Les images et extraits de textes présentés sur WikiRoom sont la propriété de Wikipédia.
